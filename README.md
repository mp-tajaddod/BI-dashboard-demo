# BI-dashboard-demo
BI dashboard demo for FMCG industry

"I have created a Power BI dashboard hosted in Microsoft Cloud, which showcases a custom template I designed for the FMCG industry. This dashboard consists of four modules: Sales, Finance, HR, and Targeting. Within the Finance module, we fully calculate the cost per unit by considering both the BOM and overhead. This allows us to see the total cost of each product. In the Targeting module, we use this cost data to set targets, showing how costs shift when scaling up sales, based on overhead adjustments. Using ABC (Activity-Based Costing), we calculate costs online in Finance and, in Targeting, we analyze how exceeding factory capacity impacts costs per unit. This dashboard has actually been implemented for three FMCG companies in Iran; we have real data, though for confidentiality reasons, we do not disclose specifics here. The demo provided is a replica of that solution. Additionally, the process follows best practices: we pull data from ERP systems, apply ETL (Extract, Transform, Load) processes, clean the data, and calculate in DAX. All of these steps are based on industry standards and official Power BI documentation.
